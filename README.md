
# Sanasto

Tämä on Aalto-yliopiston tutkimukseen perustuva epävirallinen sanasto.

Sanasto sisältää käsitteitä, määritelmiä ja synonyymejä liittyen rakennetun ympäristön digitalisaatioon, ja se on julkaistu koneluettavassa muodossa (JSON-LD).


## Lisenssi

Tämä sanasto on julkaistu lisenssillä Creative Commons Attribution 4.0 (CC BY 4.0).

Voit käyttää, muokata ja jakaa aineistoa, kunhan lähde mainitaan.

## Vastuunrajaus

Sanasto on epävirallinen julkaisu. Sen sisältö voi sisältää puutteita tai tulkinnanvaraisuuksia.

Voit antaa palautetta GitHub Issues -toiminnon kautta
